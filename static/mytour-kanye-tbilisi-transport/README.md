# Mytour Kanye West Tbilisi Transport Plan

Static Leaflet-based operations map for the Mytour group transport plan:

- Hotel group routes to Boris Paichadze Dinamo Arena
- Mytour ticket collection point near the arena
- Shared post-event coach park for Groups 2-5
- Yandex Maps links for local routing, with Google Maps fallback links

## Check Notes

The original attached HTML and JS were reviewed before deployment. The only meaningful discrepancy found was Biography Design Tbilisi:

- Booking, Tripadvisor, Georgia Travel, and other hotel listings show `15 Bezhan Kalandadze St`.
- Yandex currently shows a `Biography Tbilisi` listing at `12A Bezhan Kalandadze St`.

The deployed page keeps the official hotel-listing address but flags the mismatch so the exact driver entrance can be confirmed before operations.
